# Project Study Tracker Java

## Description
A Java console-based application to track and maintain study logs, including subjects, duration, and progress.  
It can generate summaries by date and by subject, and export study logs to CSV.

## Features
- Insert new study logs
- View all study logs
- Summary of study logs by date
- Summary of study logs by subject
- Export logs to CSV

## How to Run
1. Open terminal / command prompt in project directory
2. Compile:
           javac StudyTrackerApplication.java
3.Run:
      java StudyTrackerApplication
